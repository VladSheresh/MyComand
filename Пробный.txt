1.	Git add . – добавить все изменения
2.	Git commit –m “сообщение” – создать коммит
3.	Git push origin(ссылка на удалённый репазиторий) “название ветки” – отправить ветку в удалённый репазиторий
4.	Git branch –b “название ветки” – создать ветку
5.	Git branch - -list – список веток
6.	Git branch  -r - -list – список удалённых веток
7.	Git checkout “название ветки” – перейти в ветку 
8.	Git checkout -b “название ветки” – создать и прейти в эту ветку
9.	Git log – список коммитов
10.	Git remote – список ссылок на удалённый репазиторий (origin)
11.	Git checkout -b “название ветки” “название коммита” – создать ветку от коммита и перейти в неё
12.	Git status – посмотреть изменения 
13.	Git pull origin(ссылка на удалённый репазиторий) “название ветки” – получить изменение в ветке с удалённого репазитория
14.	Git init - создать пустой репозиторий Git или повторно инициализировать существующий
